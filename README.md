# Gym SQL Analysis Project

## Overview
This project analyzes gym members data using SQL.

## Dataset
The dataset was obtained from Kaggle and includes:
- Members data
- Workout data
- Session performance

## Data Processing
- Cleaned column names
- Split data into multiple tables (members, workouts, sessions)
- Added IDs and relationships
- Generated random names based on gender

## Analysis
- Gender distribution
- Age segmentation
- Calories burned per workout
- Advanced insights using subqueries

## Tools Used
- SQL Server
- Excel
